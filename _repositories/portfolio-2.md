A complete set of scripts for generating the PSB lattice and particle distributions for tracking studies. It supports advanced features such as multi-turn injection, injection chicane effects, perturbations, foil scattering, acceleration, machine imperfections, and dynamic tune ramps. A basic set of analysis functions is also included.

Read more on [Github](https://github.com/tprebiba/psb-xsuite-tracking).
