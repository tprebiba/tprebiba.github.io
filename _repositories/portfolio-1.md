A set of Python scripts for:
- facilitating beam dynamics analysis (fitting, statistics, FFTs, emittance deconvolution, etc.).
- scripting Machine Development (MD) studies in the CERN injectors using pyjapc, pyjapcscout and pjlsa.
- facilitating the data extraction from CERN [NXCALS](https://nxcals-docs.web.cern.ch/current/).
- loading the PSB, PS and SPS optics using cpymad.

Explore this repository on [Gitlab](https://gitlab.cern.ch/tprebiba/beam-analysis-sandbox/-/tree/master?ref_type=heads).
