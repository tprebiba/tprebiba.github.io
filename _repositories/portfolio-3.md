This application uses the k-modulation method to measure the mean-beta function at the quadrupoles of the CERN PSB and PS.

It has been used during the Run 3 PSB commissioning for measuring and correcting the injection chicane induced beta-beating and tune distortions. 

Find the source code on [Gitlab](https://gitlab.cern.ch/tprebiba/k-modulation-analysis-application-for-psb-and-ps).
